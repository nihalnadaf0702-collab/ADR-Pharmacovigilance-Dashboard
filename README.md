# ADR-Pharmacovigilance-Dashboard
Power BI dashboard analyzing adverse drug reaction reports using FAERS pharmacovigilance data.
![ADR Dashboard](dashboard_preview.png)
## Tools Used
- Power BI
- Data Modeling
- Healthcare Data Analytics

## Key Insights
- Identified the most frequently reported adverse drug reactions
- Analyzed suspect drugs associated with ADR reports
- Explored ADR distribution by gender
- Examined serious outcomes such as death and hospitalization
